# 手势控制贪吃蛇游戏

基于OpenCV和MediaPipe的手部追踪贪吃蛇游戏demo。

## 功能特点

- 使用MediaPipe进行实时手部追踪
- 食指指尖（landmark 8）控制贪吃蛇移动
- 碰撞检测：撞墙或撞到自己时游戏结束
- 实时显示分数
- 支持游戏重启

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行游戏

```bash
python snake_game.py
```

## 操作说明

1. **控制方式**：将食指移动到摄像头前，食指指尖的位置会控制贪吃蛇的移动方向
2. **重新开始**：按 `R` 键重置游戏
3. **退出游戏**：按 `Q` 键退出

## 游戏规则

- 贪吃蛇会跟随食指指尖移动
- 吃到红色食物可以增加分数和长度
- 撞到屏幕边缘或自己的身体时游戏结束

## 技术实现

- **OpenCV**：摄像头捕获和图像处理
- **MediaPipe**：手部关键点检测和追踪
- **NumPy**：数值计算和数组操作

## 注意事项

- 确保摄像头正常工作
- 保持手部在摄像头视野范围内
- 光线充足的环境下效果更好
